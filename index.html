O que é o GoblinDB?

É uma base de dados de conhecimento estruturado. Diferente de bancos tradicionais que guardam dados fragmentados, o GoblinDB guarda documentos Markdown íntegros.

Ele entende a hierarquia do seu texto, permitindo que Agentes de IA consumam apenas secções específicas ou naveguem por toda a sua rede de pensamentos.

brain/agent-task.md 
# Meta-Análise do Agente

O objetivo é resolver [[infra/setup]].

## Observações
- Memória persistente ativada.
- Links detectados: 3.

## Conclusão
A tarefa foi concluída com sucesso.

Três Caminhos para o Saber

Explore o seu conhecimento de forma Vertical, Horizontal ou em Grafo.

Busca Vertical

Consultas clássicas (Queries). Encontre documentos por tags, categorias, datas ou conteúdo específico dentro das secções.

Busca Horizontal

Hyperlinks nativos. O GoblinDB mapeia automaticamente os [[wikilinks]], permitindo saltar de um contexto para outro instantaneamente.

Navegação em Grafo

Explore conexões. Descubra como os seus documentos se relacionam e deixe os Agentes de IA mapearem o conhecimento subjacente.

GQL: A Linguagem do Conhecimento

Exemplos reais de como manipular a sua base de dados Markdown.

Criar Documento Completo 
CREATE DOCUMENT "kb/tutorial"
  TAGS ["ai", "help"]
  WITH CONTENT """
# Guia de Instalação
Siga estes passos para configurar...
Veja mais em [[installation|Setup]].
"""
Atualizar Secção Específica 
UPDATE DOCUMENT "docs/api"
  SECTION "Version"
  WITH CONTENT """
## Version
Current: 2.1.0-stable
"""
Busca Vertical (Query) 
FIND DOCUMENTS 
  WHERE TAGS CONTAINS "ai"
  AND CONTENT CONTAINS "rag"
Navegação em Grafo 
TRAVERSE FROM "concepts/ai" 
  DEPTH 3 
  RETURN LINKS

Grimoires

O GoblinDB é extensível através de Grimoires (plugins). Eles permitem, por exemplo, que a base responda via JSON-RPC para agentes que ainda não suportam o protocolo nativo.

Módulos Arcanos Ativados
# Habilitar ponte JSON-RPC
./goblindb grimoires --enable json-rpc --port 27020
